ORM Model Generator Help

NOTE: Models are automatically included !

rad ORM Model generater basically comes as an additional tool to make your life more easier by saving a lot of time to avoid directly coding model files. It generates models from database that has been selected in /includes/settings.php (which is amongst very first steps to start new radSYS project).


Generated model names are singular from table names, examples are below:

SQL Table
Model Name
users
User
countries
Country
linker_tags_videos
Linker_Tag_Video

Warning: You are recommended not to modify rad_modelizer.php, as it may result it unexpected behavior of generated models.

MySQL specific note: Instead of looking into information_schema modelizer first lists tables under selected database to prevent unexpected corruption that may lead to unconnected Models for tables that don't really exist.
